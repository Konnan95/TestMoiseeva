Тестовое задание 1 BPMN

 Выявленные противоречия и вопросы к заказчику:

1. Что происходит, если поставщик срывает сроки?  В модели есть бесконечное ожидание. Нужен таймер работы

2. Обязательна ли оценка сервиса? Если пользователь не поставит оценку, процесс зависнет? Стоит сделать её необязательной (параллельной).

3. Нет шага списания оборудования со склада. Когда оборудование передается логистике, должно быть отражение в Системе №3, иначе баланс склада будет неверным.

4. Разные типы оборудования. Для ноутбука и для серверного оборудования процедура закупки (согласование бюджета) разная, а в описании она одинаковая.

5. Неясен приоритет заявок. Все ли заявки обрабатываются в порядке очереди? От этого зависит скорость закрытия.
